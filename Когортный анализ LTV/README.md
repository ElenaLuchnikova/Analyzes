## Описание задачи
У нас есть данные о продажах интернет-магазина косметики и бытовой химии: [датасет](https://github.com/ElenaLuchnikova/Datasets/blob/main/data_store.csv) за 2023-2025 годы. 

Известно, что с января  по  июнь 2024 проводились кампании по привлечению клиентов, в том числе акция на повторную покупку с ограничением по времени - "Скидка на повторную покупку в течение недели". 

Нужно проанализировать динамику LTV (в данном случае выручку) в разрезе когорт, влияние маркетинговых активностей на LTV,  выявить лучшие когорты и дать рекомендации по дальнейшей маркетинговой политике. 

## Как было реализовано решение задачи
Я разбила клиентов по когортам на основании даты первой покупки. Затем я посчитала LTV двумя способами: общую выручку по каждой когорте (за каждый месяц) и выручку на каждого клиента в разрезе когорт (по периодам: 7, 14, 30, 60, 180, 270 и 365 дней с момента первой покупки). 
На основе получившейся таблицы я сделала выводы и предложила рекомендации, которые можно посмотреть [здесь](https://github.com/ElenaLuchnikova/Analyzes/blob/main/%D0%9A%D0%BE%D0%B3%D0%BE%D1%80%D1%82%D0%BD%D1%8B%D0%B9%20%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7%20Retention/Insights.md).
