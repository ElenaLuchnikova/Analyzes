# Описание задачи
У нас есть данные о продажах интернет-магазина косметики и бытовой химии: [датасет](https://github.com/ElenaLuchnikova/Datasets/blob/main/data_store.csv) за 2023-2025 годы. 

Известно, что с января  по  июнь 2024 проводились кампании по привлечению клиентов. Нужно проанализировать эффективность этих компаний в плане удержания клиентов, выявить лучшие когорты и дать рекомендации по дальнейшей маркетинговой политике.
