# Описание задачи
У нас есть данные о продажах интернет-магазина косметики и бытовой химии. Известно, что с января  по  июнь 2024 проводились кампании по привлечению клиентов. Нужно проанализировать эффективность этих компаний, выявить лучшие когорты и дать рекомендации по дальнейшей маркетинговой политике.
